## admin/ 技术栈

- **Vue 3.5** + **TypeScript 5.5** + **Vite 5** + **Element Plus 2.8**
- **Pinia**（user store，token 持久化 localStorage）
- **Vue Router 4**（路由守卫：无 token → /login）
- **axios**（baseURL `/admin/v1`，request 注 Bearer，response 401 → 退出 + 重定向）
- **unplugin-auto-import** + **unplugin-vue-components**（Element Plus 全局 auto-resolve）
- **dayjs**（时间戳格式化）

## 关键约定

- 时间戳：后端 unix 秒 → `dayjs(ts*1000).format('YYYY-MM-DD HH:mm:ss')`，统一收在 `src/utils/format.ts:formatTime`
- 金额：后端 cents → `formatAmount(cents)` = `¥ (cents/100).toFixed(2)`
- 权限：`role === 'super_admin'` 视为 `hasPerm()` 全通过；其余按 `perms[]` 字段判断
- 错误提示：业务 code != 0 → `ElMessage.error(msg)`；401 → 跳 /login
- 列表 loading + 空 ElEmpty + 错误 ElMessage 三态都覆盖
